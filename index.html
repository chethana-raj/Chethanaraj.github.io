Project

  • Automated Hydropinic System using IoT and Machine Learning - BTech Project

    • This project focused on the implementation of web application integrated machine learning-based smart hydroponics expert system. The proposed project with IoT consists of three phases, where the first phase implements hardware environment equipped with real-time sensors such as pH, temperature, water level, and camera module which are controlled by Raspberry Pi processor. The second phase implements the CNN Model for plant disease detection and classification and the system includes a chat bot for user interaction, addressing plant-related questions and providing details about any detected diseases. In the third phase, farmers can monitor the real-time sensor data using digital twin and plant leaf disease status using a web-based application. In this manner, the farmer can continuously track the status of his field using the website application.

    Project Link

  • Layoff Analysis - BTech Project

    • The project involved analyzing a layoff dataset sourced from Kaggle, which was initially loaded into SQL for data cleaning and exploratory data analysis (EDA). The dataset contained information on layoffs across various companies, industries, and locations, providing a comprehensive view of employment trends. Using SQL, the data was cleaned by removing duplicates, deleting unnecessary columns, and standardizing values to ensure consistency and accuracy. The cleaned dataset was then explored through EDA, where key insights were drawn regarding layoff trends, industry impacts, and regional differences. The results of the EDA were subsequently visualized using Tableau, where interactive dashboards and charts were created to present the findings in a visually compelling manner. This visualization enabled a deeper understanding of the layoff patterns, making the insights more accessible and actionable for stakeholders. The combination of SQL for data manipulation and Tableau for visualization provided a powerful approach to extracting and presenting valuable insights from the dataset.
